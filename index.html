Fixing

Fixing HTML and CSS codes are necessary when building any web project.

Debugging is partly technical and partly intuitive process that’s only mastered with practice. As you work on more projects, fixing and debugging will become more obvious.

When debugging, the first thing you need to do is to check for syntax errors.
